# Tournaments WA

## Intro

A [Yew](https://github.com/DenisKolodin/yew) Web App component : manage a game tournament.


Relevant games :

- [Connect Four](https://github.com/mothsART/connect_four)
- [Stratego](https://github.com/mothsART/stratego)
